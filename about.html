Our mission

We build simple, accessible web templates and learning resources that help developers move faster. This project started as a hands-on workshop to teach Git, GitHub Actions, and basic DevOps practices.

What we do

  • Provide clear, practical examples and exercises for version control.
  • Ship small, reusable web templates built with Bootstrap.
  • Demonstrate simple CI/CD workflows using GitHub Actions.

Our values

Clarity, reproducibility, and learning-by-doing. We favor small, testable changes and documentation that helps you get results quickly.
