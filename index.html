Git Workshop

โปรเจค Workshop สำหรับการเรียนรู้การจัดการ Source Code ด้วย Git และ GitHub Actions

เกี่ยวกับ Workshop

Workshop นี้ออกแบบมาเพื่อให้ผู้เข้าร่วมได้เรียนรู้และฝึกปฏิบัติการจัดการ Source Code อย่างมีประสิทธิภาพด้วย Git และ GitHub Actions

สิ่งที่คุณจะได้เรียนรู้

Git Basics

เรียนรู้พื้นฐานการใช้ Git เช่น commit, branch, merge และการจัดการ version control

GitHub Workflow

ทำความเข้าใจกับการทำงานร่วมกันผ่าน GitHub รวมถึง Pull Request และ Code Review

GitHub Actions

สร้างและใช้งาน CI/CD pipeline ด้วย GitHub Actions สำหรับการ automate workflow

DevOps Practices

เรียนรู้หลักการและแนวปฏิบัติที่ดีของ DevOps ในการพัฒนาและส่งมอบซอฟต์แวร์

เหมาะสำหรับ

  • นักพัฒนาซอฟต์แวร์ที่ต้องการเพิ่มทักษะการจัดการ Source Code
  • ทีมพัฒนาที่ต้องการปรับปรุงกระบวนการทำงานร่วมกัน
  • ผู้ที่สนใจเรียนรู้ DevOps และ CI/CD
